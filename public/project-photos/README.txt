Letakkan foto asli project kamu di folder ini dengan nama file berikut:
- web-movie.svg
- shopping-react-api.svg
- react-native-hotel.svg
- covid-news-react-api.svg
- portfolio-react.svg

Setelah file ditempatkan, card di Home dan Dashboard akan otomatis memakai foto asli tersebut.

Tips cepat:
- Jika ingin pakai JPG/PNG, cukup ganti path di src/utils/projectImageResolver.js
- Nama file fallback saat ini menggunakan SVG agar langsung terlihat walau belum ada foto asli.
